## Counter

## Using
- solc: v0.8.7
- web3: v4.6.0


## Setup
``` sh
npm install
npm install ganache --global
# update config with private key from ganache
```

## Run Local Blockhain(ganache)
```
npm run start
```
## Compile
```
npm run compile
```

## Deploy
```
npm run deploy
```

## Interact with smartcontract
```
node interact.js
```
